# Symfony Indonesia


## Kontribusi
Bagi yang ingin berkontribusi, silahkan clone terlebih dahulu, lalu install npm
```npm install```
atau instal secara global
```npm install -g```

Setelah itu lakukan
```bower install```

Build terlebih dahulu dengan grunt untuk melihat hasilnya
```grunt build```
